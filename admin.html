Connect Supabase Database

1 — Create a Supabase Project

Supabase gives you a free Postgres database that works from any HTML file — no server required.

  1. Go to supabase.com and sign up (free)
  2. Click "New project" → give it a name → set a password → Create
  3. Wait ~1 minute for it to provision

2 — Run This SQL

In your Supabase project: click SQL Editor in the left sidebar → paste the code below → Run.
